## 描述

面了么是一款为求职者设计的移动端面试日程管理工具。

收到面试邀请后，你可以粘贴邮件、聊天记录或选择截图，让面了么智能提取公司、岗位、面试时间、会议平台、会议链接和会议号。保存后，所有面试会按待进行、日历和历史视图清晰整理，帮助你减少遗漏、避免时间冲突，并更快找到下一场面试的信息。

主要功能：

- 智能识别面试邀请文字和截图
- 自动提取公司、岗位、时间、会议链接和会议号
- 支持自定义时区，降低跨时区面试时间误差
- 面试列表、日历视图和历史记录
- 会议号一键复制，会议链接快速打开
- 面试提醒和时间冲突提示
- 添加到系统日历
- 本地备份导出和导入

面了么默认将面试记录保存在设备本地。智能识别功能由面了么服务端提供，你无需配置第三方模型或 API Key。
